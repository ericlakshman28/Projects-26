# Projects-26

A collection of data science and machine learning projects, covering everything from a hackathon stock prediction model to coursework in machine learning and statistical hypothesis testing.

## 📁 Contents

### 📈 Stock Price Prediction (Hackathon Project)
**File:** `stock-price-prediction.ipynb`

A hackathon project exploring stock price forecasting using a range of techniques, including linear regression. Covers data preprocessing, model fitting, and evaluation of prediction accuracy against historical stock data.

### 🤖 Machine Learning Mini-Project
**File:** `ml-mini-project.ipynb`

A machine learning project applying ML techniques to a real-world dataset — covering data preparation, model training, and evaluation.

### 🧪 Statistical Hypothesis Testing Lab Report
**File:** `hypothesis-testing-lab-report.ipynb`

An experimental write-up applying statistical hypothesis testing methods to a dataset, including formulating hypotheses, running tests, and interpreting results.

### 🐍 Python Portfolio Assignments
**Files:** `python-assignment-1.ipynb`, `python-assignment-2.ipynb`

A set of Python coding assignments forming a general programming portfolio.
